>>>### git clone https://github.com/UP210553/UP210553_CPP.git
>>### 3. Enter the repository at the direction of the file with the command:
>>>### cd C:\Documents\...

>>>### "git clone https://github.com/UP210553/UP210553_CPP.git"

>>>### $ sudo apt-get update

>>>### gcc 01_BattleShip_Linux.c -o 01_BattleShip_Linux
